# FlexFit — Adaptive Workout Generator

**FlexFit** is a privacy-first, offline-first fitness app that instantly generates balanced, constraint-based workouts based on your available time, equipment, and joints to avoid.

## Key Features

- **Time-based generation** — input 5 to 60 minutes and get a perfectly timed workout
- **Equipment constraints** — choose from Bodyweight, Dumbbells, Kettlebells, Bands, or Pull-up Bar
- **Injury exclusions** — avoid Knees, Lower Back, Shoulders, Wrists, or Ankles (Premium)
- **Balanced algorithm** — Push/Pull/Legs/Core distribution, no single muscle group dominates
- **Swap exercises** — tap the swap icon on any exercise to get an equivalent alternative
- **Full-screen workout player** — animated form guidance, set/rep display, legible typography
- **Automated rest timer** — haptic alert when rest period ends
- **Skip or end early** — only completed sets are logged
- **Wakelock** — screen stays on during active workout
- **Workout history** — all sessions stored locally in SQLite (no cloud, no account)
- **Export** — download history as CSV or JSON
- **100% offline** — zero backend, zero tracking, zero subscriptions

## Monetization

- **Free tier**: Bodyweight-only workouts
- **Premium ($9.99 one-time)**: Unlocks all equipment types + injury exclusion engine
